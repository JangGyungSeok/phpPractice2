# Laravel 로그, 인증, validation관련

## 1. Laravel 내장 인증
	Laravel에서는 기본적인 인증 방식을 제공한다.
	php artisan ui:auth 키워드를 실행하면 Auth관련 Controller, routes, view 등을 자동으로 정의해준다.
	(인증 Model은 기본으로 있던 User를 사용한다.)
	인증관련 (로그인, 로그아웃, 회원가입)등의 기능을 자동으로 제공해준다.
